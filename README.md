# 🌙 Eid Mubarak - Cosmic Terminal Celebration

An interactive, animated Eid greeting page with a retro terminal experience, spectacular fireworks, and a fun Eidi (gift) portal. Perfect for sharing festive joy with friends and family.

## ✨ Features

### 🖥️ Terminal Experience
- Authentic retro terminal typing animation
- Syntax-highlighted code with JavaScript keywords
- Glassmorphism design with cosmic gradient background
- Responsive layout that works on all screen sizes

### 🎆 Visual Effects
- Dynamic fireworks particle system
- Floating star dust animations
- Glowing gradient text effects
- Smooth transitions and hover animations

### 🎁 Interactive Elements
1. **Terminal Loading**: Watch as the system boots up with festive messages
2. **Celebrate Button**: Click to reveal the Eid Mubarak greeting
3. **Fireworks Show**: Continuous particle explosions throughout the experience
4. **Wish Button**: After celebration, click to send your Eidi wishes
5. **Eidi Portal**: Professional payment/donation modal with:
   - Sadapay, Nayapay, JazzCash, EasyPaisa details
   - Cash acceptance option
   - Humorous camel-approved badge
   - Islamic greeting "تَقَبَّلَ اللهُ مِنَّا وَمِنْكُمْ"

### 📱 Mobile Responsive
- Fully optimized for all screen sizes
- Touch-friendly buttons
- Adaptive font sizes using clamp()
- Smooth scrolling on terminal
- Responsive canvas animations

## 🛠️ Technologies Used

- **HTML5** - Semantic structure
- **CSS3** - Modern styling with:
  - CSS Grid & Flexbox
  - Custom animations (@keyframes)
  - Glassmorphism effects (backdrop-filter)
  - Gradient text with background-clip
  - Responsive design with clamp()
- **JavaScript (Vanilla)** - No frameworks needed:
  - Canvas API for fireworks
  - Particle system animation
  - Typing effect simulation
  - DOM manipulation
  - Event handling
